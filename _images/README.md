These images are generated automatically from [gen.go](gen.go). To create them make sure the following is installed:

- [freeze](https://github.com/charmbracelet/freeze)
- [psudoFont Liga Mono](https://github.com/psudo-dev/psudofont-liga-mono)
- [go](https://go.dev)

Then run this command in this directory:

```bash
go run gen.go
```
